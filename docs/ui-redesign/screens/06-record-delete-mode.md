# 06 Record Delete Mode

## 参照画像

`../../image/image_if/記録_削除モード_if.png`

## 目的

通常の記録ページと同じUIのまま、整理中であることだけを明確にする。

## 構成

- 通常記録ページと同じ日付、部位チップ、検索、3列カード、下部ナビ
- 編集ボタンは完了状態
- 赤い案内: `編集モード: タップしてリストから非表示`
- 各カード左上に削除バッジ
- 追加カードは同じ位置に残し、無効/弱表示

## 表示テキスト

`編集モード: タップしてリストから非表示`, `完了`, `種目を追加`

## 状態差分

通常モードとの差分は削除バッジ、案内、完了状態、追加カード無効化のみ。

## 関連画面への追従

削除モードではカードの列数、サイズ、線画位置、MAX、種目名表示を通常記録ページから変えない。編集状態の差分だけを重ねる。

## 実装対象

`RecordPage`, `BodyPartCard`

## 未対応許容範囲

削除時の確認ダイアログは既存挙動を優先する。
