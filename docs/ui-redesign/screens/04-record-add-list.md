# 04 Record Add List

## 参照画像

`../../image/image_if/記録_種目追加_リスト_if.png`

## 目的

非表示にした種目や追加候補を素早く戻せる一覧にする。

## 構成

- ヘッダー: `種目を追加`, `閉じる`
- タブ: `リストから選択`, `カスタム作成`
- 部位チップ
- 検索バー
- 3列候補カード: 線画イラスト、部位/サブグループ、種目名、追加アイコン

## 表示テキスト

`種目を追加`, `リストから選択`, `カスタム作成`, `追加したい種目を検索`, `+追加`

## 状態差分

検索結果なし、候補なし、追加成功。

## 関連画面への追従

候補カードは記録ページの3列カードと同じサイズ感、線画、暗色面、1行種目名に揃える。MAXは追加前の候補では原則出さず、代わりにサブグループと追加アイコンを出す。

## 実装対象

`AddExerciseModal`

## 未対応許容範囲

`最近削除` タブは初期実装では追加しない。
