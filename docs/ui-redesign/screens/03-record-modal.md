# 03 Record Modal

## 参照画像

`../../image/image_if/記録モーダル_if.png`

## 目的

セット入力、前回コピー、履歴確認を同じ文脈で操作できる記録シートにする。

## 構成

- ヘッダー: 種目名、部位チップ、日付、閉じる、タイマー
- 種目サマリー: 記録カードと同じ線画、MAX、部位色アクセント
- タブ: 記録、履歴、解説
- MAX更新チェック
- セット入力行
- `セットを追加`
- 前回記録カード
- 固定CTA: `記録を完了する`

## 表示テキスト

`MAX重量を更新`, `前回の記録`, `コピーする`, `セットを追加`, `記録を完了する`

## 状態差分

筋トレ/有酸素、前回記録なし、履歴なし、保存中、入力エラー。

## 関連画面への追従

記録ページのカードで前回記録を省いた分、このモーダル内では前回記録を見つけやすくする。線画、MAX、種目名の扱いは記録カードと同じトーンにし、カードからモーダルへ移動したときに別UIへ見えないようにする。

## 実装対象

`ExerciseRecordModal`, `SetRecordForm`, `PreviousWorkoutRecordCard`, `HistoryTabContent`

## 未対応許容範囲

タイマー表示は既存タイマー状態に追従できる範囲でよい。
