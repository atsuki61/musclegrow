# 05 Record Add Custom

## 参照画像

`../../image/image_if/記録_種目追加_カスタム_if.png`

## 目的

カスタム種目を迷わず作れるフォームにする。

## 構成

- ヘッダー: `カスタム種目`, `閉じる`
- タブ: `リストから選択`, `カスタム作成`
- 入力: 種目名、部位、筋肉サブグループ、メモ
- CTA: `作成して追加`
- 確認カード: 記録カードと同じ線画/フォールバック、部位色、1行種目名

## 表示テキスト

`種目名`, `部位`, `筋肉サブグループ`, `メモ`, `作成して追加`, `作成前に確認`

## 状態差分

未入力、保存中、保存完了。

## 関連画面への追従

日本語名でカスタム登録した場合も、`トレーニング.md` の名称と一致または表記ゆれ一致するものは既存線画を自動適用する。線画がない場合は記録カードと同じフォールバック表示にする。

## 実装対象

`CustomExerciseForm`, `AddExerciseModal`

## 未対応許容範囲

テンプレートタブは初期実装では追加しない。
