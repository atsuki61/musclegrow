# 07 History Calendar

## 参照画像

`../../image/image_if/履歴_カレンダー部分_if.png`

## 目的

月間のトレーニング頻度と部位分布を一目で見る。

## 構成

- ヘッダー: `履歴`
- 部位フィルター
- 月間カレンダー
- 部位色付き日付
- 月間サマリー
- 下部ナビ: 履歴 active

## 表示テキスト

`履歴`, `月間`, `総セット`, `最多部位`, `継続`

## 状態差分

記録なし、複数部位日、部位フィルター。

## 実装対象

`HistoryPage`, `HistoryCalendar`, `BodyPartFilter`, `MultiPartDayButton`

## 未対応許容範囲

月間サマリーは取得可能なデータから段階的に追加する。
