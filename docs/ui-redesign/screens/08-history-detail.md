# 08 History Detail

## 参照画像

`../../image/image_if/履歴_履歴表示部分_if.png`

## 目的

日別の種目とセット内容を強い数字で読みやすくする。

## 構成

- 部位フィルター
- 日付見出し
- 共有、追加
- `WORKOUTS`
- 種目カードとセット行: 記録カードと同じ線画、MAX、種目名のトーンで、セット内容は下段に整理
- 下部ナビ: 履歴 active

## 表示テキスト

`WORKOUTS`, `追加`, `1RM`

## 状態差分

記録なし、削除、共有モーダル。

## 関連画面への追従

履歴の種目カードは記録ページと同じ種目認識ができるよう、線画とMAX表示のルールを共有する。履歴では前回記録ではなく、その日のセット、1RM、メモを優先する。

## 実装対象

`SessionHistoryCard`, `ExerciseCard`, `SwipeableExerciseCard`, `ShareModal`

## 未対応許容範囲

スワイプ削除の動作は既存のまま、カード見た目のみ段階的に寄せる。
