# Supabase JS クライアント

Better Auth + Drizzle ORM で Postgres に接続しているため、**現状のアプリコードからは未使用**です。

| ファイル | 用途 |
|----------|------|
| `browser-client.ts` | ブラウザ用 `createClient` |
| `server-client.ts` | Server Component / Action 用 `createServerClient` |

ローカル Postgres や CLI はリポジトリルートの `supabase/` を参照してください。
